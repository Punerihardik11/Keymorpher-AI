# Keymorpher AI  
### Gesture Based Virtual Keyboard

## Overview
Keymorpher AI is a gesture-based virtual keyboard system that allows users to type text using hand and finger movements captured through a webcam. The system uses computer vision techniques to detect hands, track finger movement, and register virtual key presses in real time.

This project is developed as a **Final Year BSc Computer Science Students** and follows **OOSE (Object-Oriented Software Engineering)** methodology for analysis and design.

---

## Project Objectives
- Detect human hand using a webcam
- Track finger movements accurately
- Display a virtual keyboard on screen
- Detect key selection based on gestures
- Register key presses and display typed text
- Ensure cross-platform compatibility (Windows & Ubuntu)

---

## Technology Stack
- **Programming Language:** Python
- **Computer Vision:** OpenCV
- **Hand Tracking:** MediaPipe
- **Numerical Processing:** NumPy
- **IDE:** Visual Studio Code
- **Version Control:** Git
